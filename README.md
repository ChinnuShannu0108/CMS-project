# CMS Project

## Overview

CMS Project is a full-stack Content Management System designed to help users create, manage, edit, and organize digital content easily. The application provides a simple and user-friendly interface for content management while ensuring secure data storage and smooth performance.

## Features

* User-friendly dashboard
* Create, edit, update, and delete content
* Secure data management
* Responsive design for different devices
* Database integration for storing content
* Fast and efficient content handling

## Technologies Used

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Project Structure

CMS-project/

* frontend/ → User Interface (UI)
* backend/ → Server-side logic and APIs

## How to Run the Project

### Frontend

1. Open the frontend folder
2. Install dependencies:
   npm install
3. Start the application:
   npm run dev

### Backend

1. Open the backend folder
2. Install dependencies:
   npm install
3. Start the server:
   npm start

## Purpose

The main goal of this project is to simplify content management by providing an easy-to-use platform where users can efficiently manage digital content without technical complexity.
